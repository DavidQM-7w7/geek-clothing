# Geek Clothing

## Descripción

Geek Clothing es una base de datos desarrollada en MongoDB para la administración de una tienda de ropa geek. El sistema permite gestionar usuarios, marcas, prendas y ventas mediante operaciones CRUD y consultas de análisis de información.

El proyecto contempla la gestión de:

- Usuarios
- Marcas
- Prendas
- Ventas

## Tecnologías utilizadas

- MongoDB
- Git
- GitHub
- Markdown

## Estructura del proyecto

```text
database/
├── geek_clothingdb.js
├── usuarios.json
├── marcas.json
├── prendas.json
└── ventas.json
```
